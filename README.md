# VSCode 단축키 정리

### 문장 순서 바꾸기
* **MAC** : `Option` + 방향키
* **Windows** : `Alt` + 방향키 

### 한 줄 지우기
* **MAC** : `Command` + `Shift` + K
* **Windows** : `Ctrl` + `Shift` + K

### 이름 바꾸기
* **MAC** : `F2`
* **Windows** : `F2`

### 줄 맞추기
* **MAC** : `Command` + K + F
* **Windows** : `Ctrl` + K + F

***

## 옵션 설정

### 자동 줄바꿈
_Settings_ 에서 `WordWrap` 설정을 **on**으로
